## [v2 구현 완료 요약]

### 구현한 기능
- **지도**: 전체화면 버튼(⛶) · 마커 클릭 → 우측 정보패널(대표 이미지+제목+유형+설명+위키링크+출처) · 유형별 범례 필터
- **사건(연표)**: 사건 클릭 → 하단 상세카드(대표 이미지+설명+위키백과 등 링크)
- **관계망**: 원 안 사진 노드(circularImage) 기본 뷰 ↔ 점 노드 전환 · 클래스 범례 필터 · centralGravity/springLength를 천천히 진동시켜 '살아있는 듯' 지속 움직임
- **언어**: 호버 시 빈도수 툴팁 · 워드클라우드/막대그래프/히트맵 3뷰 전환
- **해설**: 4개 섹션마다 '핵심 내용' 서술 + 하단 종합 해설 섹션(서술 3단락 + 요약 4카드)

### ⚠️ 강사 검토 권장 사항
1. **대표성 낮은 이미지(스탠드인)** — 완벽히 딱 맞는 PD 이미지가 없어 근접 이미지로 대체:
   - 부산(도시) → 부산항 1998 항공사진 / 정부이전 → 1950 부산 철도 / 서울환도 → 1953 서울 거리 / 김환기 → 환기미술관 건물(초상 저작권 회피) / 부산정치파동·발췌개헌 → 이승만 초상.
   실제 전시라면 부산시립·국가기록원 등에서 시대에 맞는 이미지로 교체 권장.
2. **도식(SVG) 12건** — 저작권 안전 이미지가 없어 자체 도식 사용(밀다원다방·우암동·아미동·40계단·이중섭·은지화·밀다원시대·발췌개헌·부산역전대화재·대한민국정부·피란민·헌책방).
3. **CC BY / CC BY-SA 이미지**는 UI 정보패널·CREDITS.md에 저작자·라이선스 표기(귀속 의무 이행).
4. **위키 링크**: 정전협정→영문 위키(한글 전용 문서 없음), 밀다원시대→저자 김동리 문서로 연결.
5. **워드클라우드 형태소**: 조사만 규칙 제거라 '좁은·터지자' 등 용언 어간이 일부 상위어에 섞임(교육용 허용 수준). 정밀히 하려면 형태소 분석기 필요.
6. **전체화면**: 브라우저 정책상 실제 사용자 클릭(제스처)이 있어야 작동(자동 테스트로는 검증 불가, 버튼·핸들러는 정상).

## [질의 섹션 — 온톨로지에게 묻다 (SPARQL·AI)]
강의 핵심(트리플로 정리하면 AI 자유해석과 달리 환각 없는 결과) 증명용 섹션.
- **엔진**: Oxigraph-WASM(CDN)으로 **실제 SPARQL 1.1 실행**. 스파이크로 한글 IRI 원형 처리·집계(COUNT/GROUP BY)·FILTER IN·STRSTARTS·OPTIONAL·UNION 모두 브라우저에서 동작 확인 → 폴백 불필요.
- **데이터**: 인라인 트리플→N-Triples 변환. 사건 8개 전부 date+occurredAt 부여(occurredAt이 원 트리플에 없던 사건도 EVENTS로 보강).
- **버그 수정**: 사건 노드가 이름(무공백)·원제(공백) 두 라벨을 가져 필터질의(Q4)가 중복 반환 → `ent()`가 사건은 원제 라벨만 쓰도록 수정(트리플 중복 제거).
- **질의 6종**: 경로(다단계)·전수·이웃·필터조인·집계·부재. 파라메트릭(인물/장소 선택).
- **AI 대조**: 원문만 준 실제 모델 응답을 사전 생성(백엔드/키 없음, 정직 라벨). 특히 Q2(정착지 과잉 포함)·Q3(김환기 누락)·Q6(외부지식 환각)에서 차이가 뚜렷.
- **검증**: 헤드리스 Chrome로 6종 결과 정확성·근거 배지·하이라이트(Q1 근거 4노드만 강조)·콘솔 오류 0 확인.

## [SPARQL 플레이그라운드 추가]
질의 섹션 하단에 직접 편집·실행 플레이그라운드. Oxigraph 스토어 재사용.
- textarea + 실행(Ctrl+Enter), 예제 5종, 어휘 치트시트(클래스·술어·개체), 접두어 자동 삽입.
- SELECT→표, ASK→불리언, CONSTRUCT→트리플 표. 파싱 오류는 위치까지 표시(학습용).
- 검증: 인물5행·집계25행·한글IRI질의·오류표시 모두 정상.

## [빈칸 채우기 쿼리 빌더]
플레이그라운드 상단에 드롭다운 3칸(주어·관계·목적어) 빌더. 각 칸 = 개체 또는 ?변수.
- 선택 시 SPARQL 실시간 조립(변수 칸이 SELECT 대상), '만들어 실행' → 편집창에 채우고 실행.
- 옵션: 주어 30(개체+변수), 관계 24(변수+a+19관계+3속성), 목적어 35(변수+클래스5+개체).
- 검증: 기본(피란민 settledAt ?목적어)→3행, a+Person→인물 5명, 전체변수→전체 트리플.
