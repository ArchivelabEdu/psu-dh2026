## 저작권 정책 (이미지)
- **원칙**: 실제 사진은 Wikimedia Commons에서 **퍼블릭도메인(PD)·CC0·CC-BY(-SA)** 로 확인된 것만 사용.
- 라이선스 확인 안 되거나 적합한 이미지가 없는 항목은 **직접 만든 SVG 썸네일**(유형/클래스 색상 + 이름)로 대체.
- 모든 실제 이미지의 출처·저작자·라이선스는 `CREDITS.md`에 기록.

## [v2 구현 완료 요약]

### 구현한 기능
- **지도**: 전체화면 버튼(⛶) · 마커 클릭 → 우측 정보패널(대표 이미지+제목+유형+설명+위키링크+출처) · 유형별 범례 필터
- **사건(연표)**: 사건 클릭 → 하단 상세카드(대표 이미지+설명+위키백과 등 링크)
- **관계망**: 원 안 사진 노드(circularImage) 기본 뷰 ↔ 점 노드 전환 · 클래스 범례 필터 · centralGravity/springLength를 천천히 진동시켜 '살아있는 듯' 지속 움직임
- **언어**: 호버 시 빈도수 툴팁 · 워드클라우드/막대그래프/히트맵 3뷰 전환
- **해설**: 4개 섹션마다 '핵심 내용' 서술 + 하단 종합 해설 섹션(서술 3단락 + 요약 4카드)

### 검증
- **헤드리스 Chrome**으로 전수 시각 검증(인앱 브라우저 페인은 vis-timeline 렌더 불가 → 신뢰 불가).
- 확인: 지도 마커14·정보패널·이미지 로드, 연표 8사건·위키링크, 관계망 이미지노드 29개 전부 로드(깨짐0)·뷰토글·Person필터 정확히 5개, 워드클라우드/막대20/히트맵48 렌더, 콘솔 오류 없음.

### ⚠️ 강사 검토 권장 사항
1. **대표성 낮은 이미지(스탠드인)** — 완벽히 딱 맞는 PD 이미지가 없어 근접 이미지로 대체:
   - 부산(도시) → 부산항 1998 항공사진 / 정부이전 → 1950 부산 철도 / 서울환도 → 1953 서울 거리 / 김환기 → 환기미술관 건물(초상 저작권 회피) / 부산정치파동·발췌개헌 → 이승만 초상.
   실제 전시라면 부산시립·국가기록원 등에서 시대에 맞는 이미지로 교체 권장.
2. **도식(SVG) 12건** — 저작권 안전 이미지가 없어 자체 도식 사용(밀다원다방·우암동·아미동·40계단·이중섭·은지화·밀다원시대·발췌개헌·부산역전대화재·대한민국정부·피란민·헌책방).
3. **CC BY / CC BY-SA 이미지**는 UI 정보패널·CREDITS.md에 저작자·라이선스 표기(귀속 의무 이행).
4. **위키 링크**: 정전협정→영문 위키(한글 전용 문서 없음), 밀다원시대→저자 김동리 문서로 연결.
5. **워드클라우드 형태소**: 조사만 규칙 제거라 '좁은·터지자' 등 용언 어간이 일부 상위어에 섞임(교육용 허용 수준). 정밀히 하려면 형태소 분석기 필요.
6. **전체화면**: 브라우저 정책상 실제 사용자 클릭(제스처)이 있어야 작동(자동 테스트로는 검증 불가, 버튼·핸들러는 정상).
